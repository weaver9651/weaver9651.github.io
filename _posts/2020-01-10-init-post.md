Github 을 이용해 블로그를 시작해보려고 한다. 블로그를 시작하는 데는 몇 가지 이유가 있는데 첫째로는 정보를 저장하고 공유하기 위해, 둘째로는 글을 쓰고 생각을 정리하는 시간을 갖기 위해, 그리고 세 번째로는 나를 조금씩 표현하기 위해서이다.

<br/>
여러 플랫폼 중에서 Github을 선택한 이유는 개발자로서 조금이나마 도움이 되지 않을까하는 생각과 지금까지 저장하고 필기한 공부 내용이 Markdown 형식으로 되어 있어서 Github 블로그로 이전하는 것이 편할 것 같았기 때문이다.

<br/>
꾸준한 블로그 포스팅을 통해 성장하는 기회가 되면 좋겠다.
